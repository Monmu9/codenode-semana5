# Portfolio Personal — Montserrat Muñoz Clarà

Portfolio personal desarrollado con Astro como parte de la Semana 5 de Codenode.

## 🌐 Demo en vivo

https://monmu9.github.io/codenode-semana5

## 📄 Páginas

- **Inicio** — Presentación personal con foto de perfil
- **Sobre mí** — Stack tecnológico e info personal
- **Proyectos** — Listado de proyectos con enlace a página de detalle individual

## 🛠️ Tecnologías

- Astro
- Tailwind CSS
- Markdown
- GitHub Pages
- GitHub Actions

## 🧩 Conceptos aplicados

- Componentes reutilizables con props (`ProjectCard.astro`)
- Layout compartido con navegación
- Rutas dinámicas (`[slug].astro`)
- Assets estáticos en `public/`
